## 🌐 Description de la réalisation

Le projet de **modernisation du site web d'ACTEMIUM DIJON** a été réalisé afin d'améliorer la **visibilité en ligne de l'entreprise**, d’optimiser l’**expérience utilisateur** et de **faciliter l'accès à l'information** pour les clients, partenaires et prospects. Ce projet répond à des enjeux de **communication digitale**, de **valorisation de l’image de marque**, et de **performance web**.

## 💻 Environnement technologique

- **Plateforme** : WordPress
    
- **Langages et outils** :
    
    - HTML / CSS / JavaScript
        
    - Thèmes et plugins WordPress
        
    - Outils SEO (ex : Yoast SEO)
        
    - Google Analytics
        
- **Projet réalisé en équipe**
    
- **Situation vécue (réelle)**
    

## ⚙️ Fonctionnalités et actions réalisées

- 🎨 **Personnalisation visuelle** :
    
    - Modification de thèmes WordPress selon la charte graphique d’ACTEMIUM
        
    - Intégration de modules interactifs (menus, sliders, formulaires)
        
- 🧩 **Fonctionnalités ajoutées via plugins** :
    
    - Sécurité, performance, SEO, formulaires de contact, cache...
        
- 🔍 **Optimisation SEO** :
    
    - Amélioration des balises méta, titres, URLs, vitesse de chargement
        
    - Génération de sitemap et intégration dans la Search Console
        
- 📈 **Analyse du trafic** :
    
    - Configuration de **Google Analytics**
        
    - Suivi du comportement des utilisateurs et des pages visitées
